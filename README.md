# Traffic-Light-Controller

FSM-based Traffic Light Controller design and verification using Verilog HDL.

## Description

This project implements a Traffic Light Controller using a Finite State Machine (FSM). The controller cycles through RED, GREEN, and YELLOW states based on clock timing.

## Files

* traffic_light.v – Verilog design file
* traffic_light_tb.v – Testbench file
* waveform.png – Simulation waveform

## State Sequence

RED → GREEN → YELLOW → RED

## Tools Used

* Verilog HDL
* EDA Playground
* EPWave

## Output

The controller changes traffic light states automatically according to the FSM design and timing constraints.
